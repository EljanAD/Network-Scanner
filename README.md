# Network-Scanner
This code is a Python code that will scan the given Hostname. You are able to set the start and the end ports that you want to scan in the target IP.

# How to use it
In order to use this Network-Scanner you will need (Nmap) installed in your machine. 
Next you will need to run the Python code and next you will insert your target IP or Hostname.
After providing the IP/Hostname the code will ask you to set start and end ports that you want to scan.
This code will later look for vulnerabilities in the ports set by the user.
